int Screen::init ( )

Inicializa as variaveis e a SDL.

Inicia os módulos necessários da SDL para a execução do jogo.

Returns:
0 se tudo ocorreu bem e < 0 se houve algum erro
  • Tarefas realizadas pela função.
    1. Inicia a SDL com a FLAG para iniciar tudo
    2. Inicia a tela principal com as dimensões 800x600 e 32 bits por pixel
    3. Seta o título da janela
Parameters:
[in]fTripa de bytes a ser convertida para uma struct de PDU.
Returns:
Uma referência para uma struct de PDU é retornada ou um valor NULL é passado.
Attention:
Lembrar de chamar destroy ao encerrar o jogo
